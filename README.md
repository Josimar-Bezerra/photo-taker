# Photo Take

An Ionic Angular application for taking a photo a day.

---

## About The Project

This Ionic Angular application features:

- Ability to take a photo a day
- Slideshow to visualize the photo taken
- Ability to delete photos

Built with performance, accessibility, and minimalism in mind.
